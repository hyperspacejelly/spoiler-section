Spoiler hide/show block scaffolded with Create Block tool.

== Description ==

A simple plain colored block with a hide/show button. 

Allows 3 types of blocks as InnerContent: Paragraphs, Heading, Images.

Specify the type of spoiler in a small textbox above the InnerContent

Color is customizable with wordpress' color tools.
